How to install

Prerequisite

Apache 2

$ sudo apt-get install apache2

MySQL Server

$ sudo apt-get install mysql-server

Python 2.7

$ sudo apt-get install python

Python-pip

$ sudo apt-get install python-pip

Python-mysqldb

$ sudo apt-get install python-mysqldb

Download LibreHatti

Download as .zip

Download as .tar.gz

Clone from github

Installation

  • Using cd command reach Librehatti folder
    • $ cd /path/to/Librehatti/
  • Install the required python packages in one go by
    • $ sudo pip install requirements.txt
  • Create a database for librehatti
    • $ mysql -u root -p
$ create database librehatti;
$ quit
  • Configure settings.py according to your system
    • vim /path/to/Librehatti/src/librehatti/settings.py
    Line no 15
    NAME : librehatti
USER : Your MySQL username
PASSWORD : Your MySQl password _
    Line no 47
    STATICFILES_DIRS = (
    '/path/to/LibreHatti/static',
) _
  • Go to your librehatti folder and shoot the following commands
    • $ cd /path/to/Librehatti/src
$ python manage.py migrate
$ python manage.py runserver 127.0.0.1:8090
    Open the browser with url "http://localhost:8090"
