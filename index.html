LIBREHATTI

An open source Office Automation software

How to install

Prerequisite

Apache 2

$ sudo apt-get install apache2

MySQL Server

$ sudo apt-get install mysql-server

Python 2.7

$ sudo apt-get install python

Python-pip

$ sudo apt-get install python-pip

Python-mysqldb

$ sudo apt-get install python-mysqldb

Download LibreHatti

Download as .zip

Download as .tar.gz

Clone from github

Installation

  • Using cd command reach Librehatti folder
    • $ cd /path/to/Librehatti/
  • Install the required python packages in one go by
    • $ sudo pip install requirements.txt
  • Create a database for librehatti
    • $ mysql -u root -p
$ create database librehatti;
$ quit
  • Configure settings.py according to your system
    • vim /path/to/Librehatti/src/librehatti/settings.py
    Line no 15
    NAME : librehatti
USER : Your MySQL username
PASSWORD : Your MySQl password _
    Line no 47
    STATICFILES_DIRS = (
    '/path/to/LibreHatti/static',
) _
  • Go to your librehatti folder and shoot the following commands
    • $ cd /path/to/Librehatti/src
$ python manage.py migrate
$ python manage.py runserver 127.0.0.1:8090
    Open the browser with url "http://localhost:8090"

Contribute

Hi, First of all welcome to the world of LibreHatti. We all here work together to make this little software better and more stable. This is our initiative to contribute something to the Open Source World. If you want to help us, Then here is how you can do it.

Try Installing

Installing LibreHatti is quite easy. Click here to go back to installation steps. By Installing you will get to know our software. The more you know about our software, the more easily you will be able to contribute to our little project.

Dive into code

Once you've explored all the areas of the project, you will become ready to dive in into code. We have tried to maintain our code everywhere. It'll be easy for you to read. If you face any problem, create an issue on github and we will be replying you there.

Coding Guidelines

We follow PEP Coding Guidelines. If you have worked with Python earlier then you may already be aware about this. You can lookup PEP Coding Guideline or you can read our wiki over here

Using Git

You must be well aware on how to use it. Fork this repository, make the changes you want and send us the Pull Request. We will test and talk with your code.
